Instructions

Coevery uses the modern .NET stack of technologies and MVC pattern, and wrap them in an abstraction layer optimized for data management and maintenance. The first edition has been successfully released on 2013/9/6. Later we will deliver you more fantastic user experience.

In the Preview edition, you can see five modules in the Navigation set by administrator. And Entities，fields, layouts, views can be added by yourself in the administrator settings. Following are the tips for your customization
1

Log into administrative page by clicking Admin button in the dropdown list of the admin icon. Then you will enter into a page showing all Entities and Perspectives.

2

If the inbuilt modules set by administrator can’t meet your needs, you can add a new entity by clicking Add Button. There some required information should be needed such as Display Name, Name, Record Name Label and Record Name. Record Name will be added as a system field used as a title to display on the detail page.

3

Once you add an Entity，the next step to do is adding fields for this new module. Click the Entity name in the navigation on the left side to enter into entity detail page. Add New Field Form will pop up after you click New Field button. Then you just have to follow the wizard steps to finish you field adding. Until now 11 kinds of fields can be used.

  • For Option Set field, you need to set some default value in the Picklist to select from.
  • For Reference field, you should create relationship between the two modules for data exchange.
4

The following important things to do is setting the grid list view and layout for your new module in Entity detail page. Click Add button on Views tab on the red circle position as the red circle shows left.

5

Then you will come to view detail page. Drag the fields from left to right panel you want to display in the grid list view to add them in the grid. Drag fields up and down to adjust the column order.

6

The layout of Record detail page can be either before setting list view or after it. Click Form Designer Button in the Entity Detail page. Then Form designer will be displayed as the following page. On the Layout Tab, you can group your displayed fields by adding section, Columns Width can be changed as you like. On the Fields Tab, you can decide which fields to display in the layout.

After doing those steps, do not forget to relate the Entity to a Perspective.

7

Click Manage Perspective in the Navigation to enter into Perspective Management page. First you should click Add button to add a perspective, there you need a title as the Perspective display name.

8

Click the link under the name of perspective, you will enter into perspective Detail page. Click Add Navigation to relate a specific entity to the perspective. Choose an Entity from the dropdown list and select a suitable icon from presupposed item. Finally click Save Button, your settings are successful.

9

Click Coevery icon to enter into foreground, you will see the modules you have added.
